### 3.4 Image Optimization

**For all pages:**
- **File size:** Compress to <100KB (use TinyPNG, ImageOptim)
- **Format:** WebP primary, JPEG fallback
- **Lazy loading:** `loading="lazy"` on all below-fold images
- **Alt text strategy:**
  ```
  ✅ GOOD: "EG4 18kPV hybrid inverter installed on Sacramento home roof"
  ❌ AVOID: "solar," "eg4," "image123"
  ```
- **Naming convention:** `eg4-18kpv-installation-[city]-roof-solar.webp`
- **Responsive images:** Use `srcset` for mobile/tablet optimization
  ```html
  <img 
    src="eg4-system-1200w.webp"
    srcset="eg4-system-800w.webp 800w, eg4-system-1200w.webp 1200w"
    sizes="(max-width: 768px) 100vw, 80vw"
    alt="EG4 solar system installation in [City]"
    loading="lazy">
  ```

**For case study + blog images:**
- Embed at strategic points (support copy, break up walls of text)
- Include captions (SEO + UX benefit)
- Use stock solar imagery as fallback + always use real installation photos

### 3.5 Mobile-First Indexing & Core Web Vitals

**Optimize for Key Metrics:**

| Metric | Target | Strategy |
|--------|--------|----------|
| **LCP** (Largest Contentful Paint) | < 2.5s | Lazy-load non-critical images; preload hero image & fonts |
| **FID** (First Input Delay) | < 100ms | Minimize JS; defer non-critical scripts; use code-splitting |
| **CLS** (Cumulative Layout Shift) | < 0.1 | Explicit dimensions for images/videos; avoid late-loaded elements |

**Next.js Built-Ins (Leverage):**
- `next/image` component (automatic optimization)
- `next/font` (font loading optimization)
- `next/dynamic` (code-splitting for lazy components)

**Technical Checklist:**
- [ ] Preload critical resources: `<link rel="preload" href="..." as="...">`
- [ ] Defer non-critical CSS (e.g., animations, secondary styles)
- [ ] Compress fonts: Use `font-display: swap` to prevent blank text
- [ ] Minify/compress CSS/JS: Use esbuild or Webpack
- [ ] Enable Gzip/Brotli on server
- [ ] Use CDN for static assets (Vercel automatic)

```javascript
export async function getStaticPaths() {

  };
}
```

```javascript
// pages/api/sitemap.xml.ts
export default function handler(req, res) {
  const cities = getCitiesFromCSV();
  const urls = [
    { path: '/', priority: 1.0 },
    { path: '/eg4-solar-hub', priority: 0.9 },
    ...cities.map(c => ({ path: `/cities/${c.slug}`, priority: 0.8 })),

```

```typescript
import Head from 'next/head';

export default function CityPage({ city }) {

}
```
